About SecretMessage

SecretMessage was developed as a project for a Third Year Software Development Module, Data Representation and Querying.

SecretMessage provides a service for creating confidential messages that are encrypted and decrypted on the client side. It allows users to generate a URL for a message that they have created. This URL will then be used to access and decrypt the message they have provided.

This service uses AES Encryption for each Message and provides the Decryption key to the user once the URL is generated. The decryption key is essential as the message will not be accessible without the valid key. As a safety measure, the key is not stored on the database and is bound to the user creating the URL.
